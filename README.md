*Local Setup

---Prerequisites

- Node.js (v18 or later)
- npm
- Android Studio (with Android Emulator configured) **or** Expo Go on an Android device

---Steps

1. Clone the repository

   -bash
git clone https://github.com/omparth/Recipe_Generator.git


2. Navigate to the project

   -bash
cd Recipe_Generator


3. Install dependencies

   -bash
npm install


4. Create a `.env` file in the project root

   -env
EXPO_PUBLIC_OPENROUTER_API_KEY= Sent On Whatsapp
    

5. Start the development server

  - bash
npx expo start
   

6. Run the application

- Press 'A' to launch the app on the Android Emulator, **or**
- Scan the QR code using the **Expo Go** app on an Android device.
